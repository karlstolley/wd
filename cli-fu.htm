Command-Line Fu: Basic Commands

This page will be added to as we get more sophisticated with the CLI in class.

Where Am I?

The print working directory command, pwd, will always tell you where in the file system you are: 

$ pwd
/home/username

What Files and Directories are Here?

To get a simple list of files and directories, run ls (list); for more details, run ls -l. To see hidden files and directories (those beginning with a ., like .git), run ls -la

How I Go Elsewhere?

The change directory command, cd, gets you to any directory you have read access to. Used by itself, cd returns you to your home directory. To change to another directory, use its name. For example, to change directories into your htdocs directory, run: 

$ cd htdocs

Running pwd now should output:

$ pwd
/home/username/htdocs

To move up one directory, run cd ..:

$ cd ..
$ pwd
/home/username

Making Directories and Empty Text Files

To make a directory, get to the location you wish to make it using cd, and then run the make directory command, mkdir: 

$ cd htdocs
$ mkdir css

To create an empty text file, use the touch command: 

$ cd css
$ pwd
/home/username/htdocs/css
$ touch screen.css
$ ls
screen.css

Deleting Files

Be careful. To delete a file, use the remove command, rm:

$ rm screen.css
$ ls screen.css
ls: screen.css: No such file or directory

Deleting Directories and Their Contents

Be extra careful. To delete a directory and all of its contents, use the recursive flag, -R with rm:

$ cd ..
$ rm -R css
$ ls css
ls: css: No such file or directory
